# Study Buddy

An AI-powered student performance tracking and study assistant application.

## 📖 Documentation
Detailed project documentation, including architecture, tech stack, and setup instructions, can be found in [DOCUMENTATION.md](./DOCUMENTATION.md).

## 🚀 Quick Start
1. **Install**: `npm install`
2. **Setup DB**: `npx prisma db push`
3. **Dev**: `npm run dev`

## ✨ Features
- **Performance Analytics**: Visual charts for tracking grades.
- **AI Predictions**: Intelligent final score estimations.
- **Study Recommendations**: Personalized advice for improvement.
- **Subject Management**: Organize your academic load.
